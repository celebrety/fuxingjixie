## 🔍 故障排查

### 问题 1: 留言提交失败

**症状**: Console 显示 "留言 Supabase 保存失败"

**解决方案**:
1. 检查 Supabase 表是否已创建
2. 检查 RLS 策略是否正确配置
3. 检查浏览器 Console 中的具体错误信息
4. 验证 Supabase Anon Key 是否正确

### 问题 2: 管理员界面无法加载数据

**症状**: admin.html 显示空白或错误

**解决方案**:
1. 打开浏览器 Console 查看错误
2. 确认 `CLOUD_CONFIG` 配置正确
3. 检查 Supabase 表是否有数据
4. 尝试点击 "刷新数据" 按钮

### 问题 3: 图片管理无法保存

**症状**: 上传图片后提示失败

**解决方案**:
1. 检查图片大小（base64 不能太大）
2. 建议使用 URL 链接方式而非 base64
3. 检查 `images_config` 表是否存在
4. 查看 Console 中的具体错误

## 🎯 关键改进点

### 相比 JSONBin.io 的优势

1. **并发安全**: 
   - Supabase 支持行级锁，不会出现并发冲突
   - 单条插入模式避免全量覆盖导致的数据丢失

2. **响应速度**:
   - Supabase REST API 响应更快
   - 全球 CDN 加速

3. **数据可靠性**:
   - 自动备份
   - 事务支持
   - 数据一致性保证

4. **扩展性**:
   - 支持复杂查询
   - 支持分页
   - 支持实时订阅

5. **离线支持**:
   - 保留了 localStorage 离线缓存
   - 保留了重试队列机制
