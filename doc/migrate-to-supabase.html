🚀 数据迁移工具

从 JSONBin.io / localStorage 迁移到 Supabase

⚠️ 重要提示：
1. 迁移前请确保已在 Supabase 中执行了数据库表结构脚本（supabase-schema.sql）
2. 本工具会从浏览器 localStorage 读取数据并导入到 Supabase
3. 如果您的数据在 JSONBin.io 上，请先在浏览器中访问一次 admin.html 让数据同步到 localStorage

📋 迁移步骤

1. 打开浏览器开发者工具（F12）

2. 访问复兴机械厂官网的 admin.html 页面

3. 等待页面加载完成，确保数据已同步到 localStorage

4. 点击下方的「开始迁移」按钮
