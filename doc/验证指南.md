## 🔍 故障排查

### 问题 1: 留言提交后 Supabase 中没有数据

**可能原因：**
1. 数据库表未创建
2. RLS 策略配置错误
3. Supabase Key 不正确
4. 网络请求失败

**排查步骤：**
```javascript
// 1. 检查 Console 是否有错误
// 2. 检查 Network 面板中的请求状态
// 3. 手动测试 API：
fetch('https://sjpkjfpaocswveydbxvc.supabase.co/rest/v1/messages', {
  method: 'GET',
  headers: {
    'apikey': 'sb_publishable_RbIuhqGsWbNzfNE-cXiZNA_7BR1_1fM',
    'Authorization': 'Bearer sb_publishable_RbIuhqGsWbNzfNE-cXiZNA_7BR1_1fM'
  }
}).then(r => r.json()).then(d => console.log(d));
```

### 问题 2: 图片配置无法保存

**可能原因：**
1. images_config 表不存在
2. 数据格式不正确
3. 网络问题

**排查步骤：**
```javascript
// 检查表是否存在
fetch('https://sjpkjfpaocswveydbxvc.supabase.co/rest/v1/images_config', {
  method: 'GET',
  headers: {
    'apikey': 'sb_publishable_RbIuhqGsWbNzfNE-cXiZNA_7BR1_1fM',
    'Authorization': 'Bearer sb_publishable_RbIuhqGsWbNzfNE-cXiZNA_7BR1_1fM'
  }
}).then(r => {
  console.log('Status:', r.status);
  return r.json();
}).then(d => console.log('Data:', d));
```

### 问题 3: 数据不同步

**可能原因：**
1. 本地缓存未更新
2. Supabase 查询缓存
3. 网络延迟

**解决方案：**
```javascript
// 清除本地缓存
localStorage.removeItem('fx_messages');
localStorage.removeItem('fx_images');

// 强制刷新页面（Ctrl+Shift+R 或 Cmd+Shift+R）
```
