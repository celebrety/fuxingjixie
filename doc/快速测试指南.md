为了让手机能访问，建议启动本地服务器：

```bash

cd /Users/celebrety/IdeaProjects/fuxingjixie/fuxingjixie
```

### 方法 2: 使用 Node.js

```bash
# 安装 http-server（首次需要）
npm install -g http-server

# 启动服务器
http-server -p 8080
```

### 方法 3: 使用 VS Code

1. 安装 "Live Server" 扩展
2. 右键 `index.html`
3. 选择 "Open with Live Server"

### 访问方式

启动后，在同一网络下的设备访问：
- 电脑：`http://localhost:8080/index.html`
- 手机：`http://你的电脑IP:8080/index.html`

查看电脑IP：
```bash
# macOS
ifconfig | grep "inet "

# 通常是 192.168.x.x 或 10.x.x.x
```

## 🐛 常见问题

### Q: 提交留言后看不到？

**检查步骤：**
1. 打开浏览器控制台（F12）
2. 查看是否有错误信息
3. 检查 `isCloudConfigured()` 是否返回 `true`
4. 检查网络连接是否正常

### Q: 提示"云端服务未配置"？

**原因：** 配置没有正确加载

**解决：**
1. 刷新页面（Cmd+R 或 Ctrl+F5 强制刷新）
2. 清除浏览器缓存
3. 检查代码中 CLOUD_CONFIG 是否正确

### Q: H5 导航还是不能滚动？

**解决：**
1. 清除浏览器缓存
2. 确保用手机浏览器打开（不是桌面浏览器）
3. 检查控制台是否有 JavaScript 错误
