## 🔧 需要您完成的配置（约10分钟）

### 步骤 1: 注册 JSONBin.io 账号

1. 访问 https://jsonbin.io
2. 点击 **"Sign Up"** 注册账号
   - 支持 GitHub 账号登录（推荐）
   - 支持 Google 账号登录
   - 也支持邮箱注册
3. 登录后，进入 Dashboard

### 步骤 2: 获取 API Key

1. 在 Dashboard 页面，找到 **"API Keys"** 部分
2. 复制您的 **X-Master-Key**（类似：`$2a$10$xxxxxxxxxxxxxxxxxxxxxx`）
3. 保存这个 Key，后面会用到

### 步骤 3: 创建留言数据仓库（Bin）

#### 方法A：使用网页界面（推荐）

1. 点击 Dashboard 上的 **"+ Create Bin"** 按钮
2. 在编辑器中输入：
   ```json
   {
     "messages": []
   }
   ```
3. 点击 **"Create"** 按钮
4. 复制 Bin ID（在 URL 中或页面顶部显示，类似：`65xxxxxxxxxxxxxx`）
5. 保存这个 ID

#### 方法B：使用命令行

```bash
curl -X POST https://api.jsonbin.io/v3/b \
  -H "Content-Type: application/json" \
  -H "X-Master-Key: 你的API_KEY" \
  -d '{"messages": []}'
```

返回结果中的 `metadata.id` 就是 Bin ID。

### 步骤 4: 创建图片配置仓库（Bin）

重复步骤3，但这次输入：
```json
{
  "images": {}
}
```

同样保存返回的 Bin ID。

### 步骤 5: 修改代码配置

现在您需要修改两个文件中的配置：

#### 修改 index.html

1. 打开 `index.html` 文件
2. 搜索 `CLOUD_CONFIG`（大约在第 5443 行）
3. 找到以下代码：

```javascript
const CLOUD_CONFIG = {
    // ⚠️ 重要：请替换为您自己的 JSONBin.io 配置
    // 注册地址：https://jsonbin.io
    API_KEY: 'YOUR_JSONBIN_API_KEY',  // 您的 API Key
    MESSAGES_BIN_ID: 'YOUR_MESSAGES_BIN_ID',  // 留言数据 Bin ID
    IMAGES_BIN_ID: 'YOUR_IMAGES_BIN_ID'  // 图片配置 Bin ID
};
```

4. 替换为您的实际值：

```javascript
const CLOUD_CONFIG = {
    API_KEY: '$2a$10$你的实际APIKey',  // 替换这里
    MESSAGES_BIN_ID: '65你的留言BinID',    // 替换这里
    IMAGES_BIN_ID: '65你的图片BinID'       // 替换这里
};
```

#### 修改 admin.html

1. 打开 `admin.html` 文件
2. 搜索 `CLOUD_CONFIG`（大约在第 334 行）
3. 同样替换为您的实际值（与 index.html 保持一致）

## ⚠️ 重要提示

### 1. 两个文件必须使用相同的配置

`index.html` 和 `admin.html` 中的 `CLOUD_CONFIG` 必须完全一致，否则无法同步数据。

### 2. 数据安全

- API Key 会暴露在前端代码中
- 对于小型项目这是可接受的
- 如果需要更高安全性，建议使用 Cloudflare Workers 作为代理

### 3. 免费额度

JSONBin.io 免费额度：
- 100,000 次请求/月
- 100 个 Bin
- 每个 Bin 最大 100KB

对于您的项目：
- 预计每天 50 条留言 = 100 次请求
- 预计每月约 3,000 次请求
- **完全在免费额度内**

### 4. 离线缓存

即使不配置云端服务，代码也能正常工作：
- 数据会保存在本地 localStorage
- 同设备访问没有问题
- 但跨设备无法同步

## 🐛 常见问题

### Q1: 提交留言后管理后台看不到？

**检查清单：**
1. 两个文件的 `CLOUD_CONFIG` 是否一致？
2. 是否正确替换了所有三个配置项？
3. 浏览器控制台是否有错误信息？
4. 网络连接是否正常？

**调试方法：**
打开浏览器控制台（F12），查看：
- 是否有 "云端服务未配置" 的警告
- 是否有网络请求失败的错误
- 留言保存的日志信息

### Q2: H5 导航还是不能滚动？

**解决方法：**
1. 清除浏览器缓存
2. 确保修改了 HTML 中的 onclick 属性（添加 `,event` 参数）
3. 检查浏览器控制台是否有 JavaScript 错误

### Q3: 图片上传失败？

**可能原因：**
1. 图片太大（超过 2MB）
2. JSONBin.io 存储空间已满
3. 网络连接问题

**解决方法：**
- 压缩图片后再上传
- 使用图片外链（推荐）
- 检查 JSONBin.io Dashboard 的使用量

### Q4: 能否不使用云端服务？

**可以！** 代码已经做了降级处理：
- 如果不配置云端，会自动使用 localStorage
- 同设备使用没有问题
- 只是无法跨设备同步
