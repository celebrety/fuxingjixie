# 复兴机械厂网站

> 基于纯前端技术的企业展示网站，支持跨设备数据同步和管理后台

## 📖 项目简介

复兴机械厂网站是一个现代化的企业展示平台，包含客户访问页面和管理后台两个部分。项目采用纯前端技术栈（HTML + CSS + JavaScript），通过 JSONBin.io 云端服务实现跨设备数据同步，无需自建服务器。

### 核心特性

✅ **跨设备数据同步** - 手机提交留言，电脑管理后台即时查看  
✅ **云端图片管理** - 管理后台配置图片，客户页面实时更新  
✅ **H5 移动端优化** - 响应式设计，底部导航平滑滚动  
✅ **离线缓存保护** - 网络异常时自动降级到本地存储  
✅ **零服务器成本** - 使用 JSONBin.io 免费服务，无需运维

## 🚀 快速开始

### 1. 本地预览

直接双击打开 HTML 文件即可在浏览器中预览：

```bash
# 客户访问页面
open index.html

# 管理后台
open admin.html
```

### 2. 启动本地服务器（推荐）

为了让手机等设备能够访问，建议启动本地 HTTP 服务器：

**使用 Python 3：**
```bash
cd /path/to/fuxingjixie
python3 -m http.server 8080
```

**使用 Node.js：**
```bash
npm install -g http-server
http-server -p 8080
```

**使用 VS Code：**
1. 安装 "Live Server" 扩展
2. 右键 `index.html` → "Open with Live Server"

启动后访问：
- 电脑端：`http://localhost:8080/index.html`
- 手机端：`http://你的电脑IP:8080/index.html`

### 3. 管理后台登录

- **访问地址：** `admin.html`
- **默认密码：** `fuxing@2024`
- **功能模块：**
  - 📊 数据概览 - 留言统计和最新动态
  - 📨 留言管理 - 查看、处理、导出客户留言
  - 🖼️ 图片管理 - 上传和配置网站图片
  - ⚙️ 系统设置 - 修改管理密码

## ⚙️ 云端配置

项目默认使用 JSONBin.io 作为云端数据存储服务。如需启用跨设备同步功能，请按以下步骤配置：

### 配置步骤

1. **注册 JSONBin.io 账号**
   - 访问 https://jsonbin.io
   - 使用 GitHub/Google 账号或邮箱注册

2. **获取 API Key**
   - 在 Dashboard 找到 "API Keys" 部分
   - 复制 X-Master-Key（格式：`$2a$10$xxxxxxxxx`）

3. **创建数据仓库（Bin）**
   - 留言数据 Bin：`{"messages": []}`
   - 图片配置 Bin：`{"images": {}}`
   - 记录两个 Bin 的 ID

4. **修改配置文件**
   
   在 `index.html` 和 `admin.html` 中找到 `CLOUD_CONFIG`（约在第 5439 行和第 336 行）：

   ```javascript
   const CLOUD_CONFIG = {
       API_KEY: '你的API_KEY',
       MESSAGES_BIN_ID: '留言Bin_ID',
       IMAGES_BIN_ID: '图片Bin_ID'
   };
   ```

5. **测试验证**
   - 手机提交留言 → 电脑后台查看
   - 电脑配置图片 → 手机刷新显示

详细配置步骤请参考：[doc/配置指南.md](doc/配置指南.md)

## 🛠️ 技术栈

- **前端框架：** 原生 HTML5 + CSS3 + JavaScript (ES6+)
- **云端存储：** JSONBin.io (REST API)
- **数据格式：** JSON
- **响应式设计：** CSS3 Media Queries
- **图片处理：** Canvas API（自动压缩）
- **本地缓存：** localStorage（离线降级）

## 📊 数据流架构

```
┌─────────────┐         ┌──────────────┐         ┌─────────────┐
│  index.html │ ◄─────► │  JSONBin.io  │ ◄─────► │  admin.html │
│  (客户页面)  │  REST API│  (云端数据库) │  REST API│  (管理后台)  │
└─────────────┘         └──────────────┘         └─────────────┘
     手机/电脑                  云端                   电脑/手机

         └─────────────> localStorage <─────────────────┘
                   (离线缓存/降级保护)
```

## 💰 成本说明

### JSONBin.io 免费额度

- **API 请求：** 100,000 次/月
- **数据仓库：** 100 个 Bin
- **单 Bin 容量：** 100 KB
- **预估使用量：** 约 3,000 次请求/月（每天 50 条留言）

**结论：** 免费额度完全够用，无需付费升级。

## ⚠️ 注意事项

1. **配置一致性**
   - `index.html` 和 `admin.html` 的 `CLOUD_CONFIG` 必须完全一致

2. **图片大小限制**
   - 上传的图片会自动压缩（最大宽度 800px，质量 70%）
   - 建议上传前自行压缩，单个图片不超过 500KB

3. **数据安全**
   - API Key 会暴露在前端代码中
   - 对于小型项目这是可接受的
   - 如需更高安全性，建议使用 Cloudflare Workers 代理

4. **离线使用**
   - 未配置云端服务时，数据仅保存在本地 localStorage
   - 同设备访问正常，但无法跨设备同步
